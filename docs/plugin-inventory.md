# Плагины и назначение

## Входит в репозиторий

- `matreshka-master-core`
  - кастомные сущности проекта
  - админские настройки
  - метабоксы главной
  - формы захвата
  - лиды
  - интеграционные точки

## Ставится bootstrap-скриптом

- `WooCommerce`
  - каталог
  - карточка товара
  - корзина / checkout
  - архитектура для подключения оплаты

- `Polylang`
  - языковая структура сайта
  - переводы страниц, меню и строк

- `Rank Math SEO`
  - SEO-метаданные
  - sitemap
  - техподготовка к продвижению

- `WP Mail SMTP`
  - нормальная доставка писем из форм

## Рекомендуется для production

- `Polylang for WooCommerce` или `WPML + WooCommerce Multilingual`
  - полноценный перевод каталога WooCommerce

- production payment gateway plugin
  - зависит от выбранной платёжной системы
