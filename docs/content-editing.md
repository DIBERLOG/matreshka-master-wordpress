# Как редактировать контент

## Главная страница

1. Открыть страницу, назначенную статической главной.
2. Использовать кастомные метабоксы:
   - Hero
   - Sections
   - Elite
   - Workshop
   - Contact section
3. Для скрытия секций использовать чекбоксы видимости.

## Витрины

1. Открыть раздел `Showcases`.
2. Создать или отредактировать карточку.
3. Назначить категорию:
   - `prints`
   - `corporate`
   - `portrait`
4. Загрузить изображения:
   - front / back
   - before / after

## FAQ

1. Открыть раздел `FAQ`.
2. Каждая запись = один вопрос и ответ.
3. Порядок регулируется `menu order`.

## Каталог

1. Открыть `Products`.
2. Добавлять товары, категории, цены, галереи и описания стандартными средствами WooCommerce.
3. Для мультиязычности каталога использовать production multilingual workflow.

## Глобальные настройки

Открыть пункт `Matreshka Master`.

Там редактируются:

- контакты
- соцсети и мессенджеры
- юридические ссылки
- URL каталога
- ordering guide
- email получателя заявок
- success message форм
- theme skin
- webhook-поля интеграций
