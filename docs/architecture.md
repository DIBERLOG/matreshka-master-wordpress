# Архитектура проекта

## Базовый стек

- WordPress как CMS
- кастомная тема `matreshka-master`
- кастомный плагин `matreshka-master-core`
- WooCommerce для каталога и архитектуры оплаты
- Polylang-ready мультиязычность

## Что лежит в теме

- премиальный фронтенд
- шаблоны главной страницы
- хедер, футер, контентные шаблоны
- стили каталога WooCommerce
- анимации, карусели, модалка форм, mobile-nav

## Что лежит в core-плагине

- CPT `mm_showcase` для витрин
- taxonomy `mm_showcase_category` для группировки витрин
- CPT `mm_faq` для FAQ
- CPT `mm_lead` для всех входящих заявок
- глобальные настройки проекта
- метабоксы главной страницы
- обработчик форм и точки интеграции
- сидирование демо-контента

## Контентная модель

- Главная:
  - одна статическая страница
  - контент редактируется через кастомные метабоксы
- Витрины:
  - отдельные записи
  - категории `prints`, `corporate`, `portrait`
- FAQ:
  - отдельные записи
- Каталог:
  - нативные WooCommerce products
- Лиды:
  - отдельные приватные записи в админке

## Мультиязычность

- архитектура готова для русского, английского и китайского
- языковой переключатель встроен в тему
- тема умеет работать с Polylang и WPML
- контент главной страницы переводится через переводы самих страниц
- для полноценного перевода WooCommerce-каталога нужен production add-on:
  - `Polylang for WooCommerce`, или
  - `WPML + WooCommerce Multilingual`

## Интеграции

- формы сохраняются в WordPress
- формы могут уведомлять по email
- Telegram поддержан напрямую через bot token + chat id
- Bitrix / WhatsApp / MAX / другие системы подключаются через webhook URL без переписывания шаблонов
