# Плагины и зависимости

## Кастомный плагин

### Matreshka Master Core

Назначение:

- контентные типы проекта
- метабоксы главной
- формы и лиды
- глобальные настройки
- demo-seed
- интеграционные точки

## Публичные плагины

### WooCommerce

Используется для:

- каталога
- карточек товаров
- цен
- будущей онлайн-оплаты

### Polylang

Используется для:

- языков сайта
- перевода страниц
- языкового переключателя

Важно:

Для полноценного мультиязычного каталога нужен production add-on для WooCommerce.

### Rank Math SEO

Используется как SEO-слой:

- мета-теги
- sitemap
- Open Graph
- базовая SEO-оптимизация

### WP Mail SMTP

Используется для подключения нормальной отправки писем после получения реальных SMTP-доступов.

## Что может понадобиться дополнительно

- `Polylang for WooCommerce`
- или `WPML + WooCommerce Multilingual`
- конкретный платёжный модуль под нужного провайдера
- CRM-модуль, если заказчик попросит готовую интеграцию через официальный плагин
