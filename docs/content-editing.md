# Редактирование контента

## Главная страница

1. Открой `Pages`
2. Найди страницу `Главная`
3. Открой её в редакторе
4. Прокрути вниз до `Meta Boxes`

Там редактируются:

- Hero
- заголовки и тексты секций
- кнопки
- блок «Для элиты»
- блок о мастерской
- контактный блок
- видимость секций

## Витрины

1. Открой раздел `Витрины`
2. Выбери нужную карточку
3. Обнови:
   - название
   - описание
   - изображения
   - перед / оборот
   - до / после
   - категорию витрины

Категории витрин:

- матрёшки с принтами
- корпоративные подарки
- портретные матрёшки

## FAQ

1. Открой `FAQ`
2. Каждая запись — это один вопрос и один ответ
3. Меняй порядок при необходимости через порядок записей или дату / menu order

## Каталог

1. Открой `Products`
2. Создай или открой товар
3. Обнови:
   - название
   - цену
   - галерею
   - краткое описание
   - полное описание
   - категории

Если каталог будет мультиязычным, после установки production multilingual add-on нужно отдельно поддерживать переводы товаров.

## Заявки

1. Открой раздел `Заявки`
2. Просматривай входящие лиды
3. Используй данные для ручной обработки или CRM-интеграции

## Глобальные настройки

1. Открой `Matreshka Master`
2. Здесь можно менять:
   - контакты
   - ссылки на соцсети
   - юридические ссылки
   - общие проектные настройки
   - интеграционные endpoint-поля

## Языки

1. Открой `Languages`
2. Управляй русской, английской и китайской версиями
3. Для корректного мультиязычного каталога настройка WooCommerce делается отдельно через production add-on

## Что лучше не менять без разработчика

- код темы
- код плагина
- WooCommerce templates
- структуру metabox-ключей
- системные настройки Docker / bootstrap
